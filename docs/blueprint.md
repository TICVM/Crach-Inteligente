# **App Name**: Crachá Inteligente

## Core Features:

- Registro Manual de Alunos: Permite o registro individual de alunos, inserindo nome, turma e carregando uma foto, que será armazenada no banco de dados e no armazenamento de arquivos.
- Importação em Massa de Alunos: Suporta a importação de dados de alunos (nome, turma) via arquivo Excel e associação com fotos locais, salvando tudo no banco de dados.
- Gestão de Alunos Cadastrados: Uma interface para visualizar, editar e excluir os dados e fotos dos alunos já registrados.
- Personalização de Fundo do Crachá: Funcionalidade para carregar e definir uma imagem personalizada que será utilizada como fundo para todos os crachás gerados.
- Geração de Lema/Slogan de IA: Uma ferramenta de IA que gera lemas ou slogans curtos e inspiradores, personalizados para estudantes, a serem incluídos nos crachás.
- Geração de Crachás em PDF: Gera um documento PDF com todos os crachás dos alunos, otimizado para impressão em formato de múltiplos crachás por folha.
- Opção de Impressão Direta: Facilita o envio direto dos crachás gerados para a impressora do usuário, sem necessidade de download prévio do PDF.

## Style Guidelines:

- Esquema de cores claro. A cor primária é um azul profundo e confiável (#2365BF), transmitindo profissionalismo e clareza, ideal para um ambiente educacional.
- A cor de fundo será um azul acinzentado muito claro (#F2F5F7), mantendo a harmonia com o azul primário para uma interface limpa.
- Como cor de destaque, um índigo/violeta escuro (#2616A6) para elementos interativos e chamadas para ação, oferecendo contraste marcante e um toque de sofisticação.
- A fonte recomendada é 'PT Sans' (sans-serif), escolhida por sua legibilidade em diferentes tamanhos e seu toque humanista, que transmite modernidade e acessibilidade, ideal para textos de formulário e no próprio crachá.
- Utilizar ícones claros e objetivos, preferencialmente em estilo outline, para guiar o usuário intuitivamente pelas funções do sistema.
- Layout responsivo e com design de cartão para cada seção de funcionalidade, proporcionando um fluxo de trabalho organizado e visualmente segmentado em qualquer dispositivo.
- Pequenas animações de transição para carregamento de dados e confirmação de ações, garantindo uma experiência do usuário suave e interativa.
